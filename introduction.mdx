# 使用 OneDrive


## Windows端

软件的默认数据路径在> C:\Users\用户名\AppData\Roaming\MyLibrary

也可以点击下方设置处的按钮跳转到数据路径下

//图片1

只需要在「设置-数据-数据路径」处将OneDrive的路径复制进去即可。

请注意，此处必须是OneDrive根目录下MyLibrary文件夹，名称、位置都不可更改。

例如，你的 OneDrive 的目录是 D:\OneDrive ，那么此处的路径是 D:\OneDrive\MyLibrary\ 

之后只需要注意，不要在两个端同时编辑数据即可。

## 移动端

点击左下角「设置图标」点击同步按钮进入同步页面。

初次进入同步页会需要登录 OneDrive 账号

同步页面的最上面有一个同步提醒的开关。

开启后，你新增或者编辑数据后，左下角的「更多按钮」会出现绿色的小圆点提醒你需要进行同步。

同步页另外另外上下两个部分，一个是将 OneDrive 的数据下载到本地 ，另是将本地数据上传到 OneDrive。

同样需要注意， **不要在两个端同时编辑数据。**

以及修改数据后及时同步上云端。

# 手动同步

## Windows

你可以使用任意同步盘(例如坚果云、阿里云盘)，和上面一样将数据路径设置到同步盘处。

或者将数据库文件发送到任意可在手机端访问的应用，例如：微信、QQ等等。

数据库文件路径：在 数据路径 Database\main.db

## 移动端

在移动端APP中下载文件，并选择在 MyLibrary 打开
